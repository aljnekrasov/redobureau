# Fable Animation — Liquid 001

3D-анимация в стиле главной страницы, но с другим эффектом: вместо статичного
логотипа — «жидкий металл», непрерывно морфящийся хромированный блоб.

Что взято с главной (`src/js/helpers/utility/threedthings.js`):

- то же психоделическое небо — сфера с текстурой `assets/models/11.jpg`;
- живые отражения через `CubeCamera` (объект — зеркало окружения);
- та же камера: плавно следует за мышью/тачем, те же коэффициенты `kX`/`kY`
  и изинг `0.05`, та же формула `z = 14 · height / width`;
- белый/чёрный фон по `prefers-color-scheme`, подписи Helvetica в углах.

Что нового:

- форма — икосаэдр, деформируемый 3D-симплекс-шумом в каждом кадре
  (двухоктавный fBm по направлениям вершин — без швов);
- блоб реагирует на скорость курсора: чем быстрее двигаешь мышь,
  тем сильнее и быстрее «кипит» поверхность (энергия затухает за пару секунд);
- медленное вращение блоба и едва заметный дрейф неба, чтобы отражения
  текли даже в покое.

## Запуск

Нужен любой локальный сервер (ES-модули и текстуры не работают с `file://`),
Three.js подтягивается с CDN (jsdelivr), поэтому нужен интернет.

Через дев-сервер сайта (из корня репозитория):

```
yarn start          # или: php -S localhost:8000 kirby/router.php
```

и открыть <http://localhost:8000/fableanimation/index.html>.

Либо любым статическим сервером из корня репозитория:

```
python3 -m http.server 8001
# → http://localhost:8001/fableanimation/index.html
```

Если сервер запущен из самой папки `fableanimation` (текстура
`../assets/models/11.jpg` недоступна), автоматически включается
процедурный фолбэк неба в той же палитре.

Для превью в Claude Code есть конфиг `.claude/launch.json`
(php-сервер на порту 8123).

## Ручки для настройки (main.js)

- `BLOB_RADIUS`, `BLOB_DETAIL` — размер и детализация сетки;
- `BASE_AMP` — базовая амплитуда «жидкости» (0.24);
- `FREQ_1`, `FREQ_2` — масштаб волн шума;
- `energy * 0.16` в `updateBlob()` — насколько мышь усиливает кипение;
- `dt * 0.12` — скорость вращения блоба.
