# Fable Animation — Liquid 001

3D-анимация в стиле главной страницы, но с другим эффектом: вместо статичного
логотипа — «жидкий металл», непрерывно морфящийся хромированный блоб.

Что взято с главной (`src/js/helpers/utility/threedthings.js`):

- то же психоделическое небо — сфера с текстурой `assets/models/11.jpg`;
- живые отражения через `CubeCamera` (объект — зеркало окружения);
- та же камера: плавно следует за мышью/тачем, те же коэффициенты `kX`/`kY`
  и изинг `0.05`, та же формула `z = 14 · height / width`;
- белый/чёрный фон по `prefers-color-scheme`, подписи Helvetica в углах.

Что нового:

- форма — икосаэдр, деформируемый 3D-симплекс-шумом в каждом кадре
  (двухоктавный fBm по направлениям вершин — без швов);
- блоб реагирует на скорость курсора: чем быстрее двигаешь мышь,
  тем сильнее и быстрее «кипит» поверхность (энергия затухает за пару секунд);
- медленное вращение блоба и едва заметный дрейф неба, чтобы отражения
  текли даже в покое;
- переключатель неба внизу по центру: **Waves** (11.jpg с главной) ⇄
  **Velvet** — процедурное зернистое сине-бирюзовое «полотно» с мягкими
  диагональными бликами (генерируется на канвасе, см. `makeVelvetSky()`).

## Запуск

Нужен любой локальный сервер (ES-модули и текстуры не работают с `file://`),
Three.js подтягивается с CDN (jsdelivr), поэтому нужен интернет.

Через дев-сервер сайта (из корня репозитория):

```
yarn start          # или: php -S localhost:8000 kirby/router.php
```

и открыть <http://localhost:8000/fableanimation/index.html>.

Либо любым статическим сервером из корня репозитория:

```
python3 -m http.server 8001
# → http://localhost:8001/fableanimation/index.html
```

Если сервер запущен из самой папки `fableanimation` (текстура
`../assets/models/11.jpg` недоступна), автоматически включается
процедурный фолбэк неба в той же палитре.

Для превью в Claude Code есть конфиг `.claude/launch.json`
(php-сервер на порту 8123).

## Ручки для настройки (main.js)

- `BLOB_RADIUS`, `BLOB_DETAIL` — размер и детализация сетки;
- `BASE_AMP` — базовая амплитуда «жидкости» (0.24);
- `FREQ_1`, `FREQ_2` — масштаб волн шума;
- `energy * 0.16` в `updateBlob()` — насколько мышь усиливает кипение;
- `dt * 0.12` — скорость вращения блоба.

## Мудборды портфолио (mood1–mood15)

Пятнадцать вариантов одной идеи: 30 работ с redobureau.com/work (по две с
каждого опубликованного проекта) на одной странице, без единой буквы. Картинки
лежат локально в `moodassets/` (≤900px, ~2 МБ на всё), данные и общий код
(сид-PRNG, точечная навигация, `loadAll`/`coverDraw` для canvas/WebGL) —
в `moodworks.js`. Между вариантами — 15 точек внизу (у экспериментальных 11–15
обведённые), стрелки ←/→ и клавиши 1–9.

### DOM / вёрстка (mood1–mood10)

- `mood1.html` — **Galds**: стол с разбросанными отпечатками; сам дрейфует,
  таскается с инерцией, у каждого отпечатка своя глубина параллакса.
- `mood2.html` — **Straumes**: стена под −8°, пять бесконечных вертикальных
  рек с разными скоростями; колесо/драг разгоняет все реки по их течению.
- `mood3.html` — **Orbīta**: три эллиптические орбиты (8/10/12 работ),
  вращаются в противофазе; драг крутит систему, колесо — зум.
- `mood4.html` — **Dziļums**: полёт сквозь 3D-коридор работ (CSS perspective);
  колесо/драг — газ, мышь подруливает взгляд, коридор зациклен.
- `mood5.html` — **Režģis**: строгий квилт на бумаге 8×6; каждые ~4 с плитки
  меняются местами (FLIP), каждый четвёртый такт — полная перекройка,
  клик — перекроить сейчас.
- `mood6.html` — **Bezgalība**: бесконечный холст без краёв; тянется во все
  стороны с инерцией, плитки рециклятся через виртуализованный пул.
- `mood7.html` — **Gaisma**: тёмная комната, курсор-фонарик выхватывает работы
  из мрака (CSS-маска), стена дышит лёгким параллаксом.
- `mood8.html` — **Peldēšana**: работы плавают как в аквариуме с мягкой
  физикой; курсор толкает воду, они разбегаются и возвращаются к дрейфу.
- `mood9.html` — **Kārtis**: центральная колода; верхнюю карту сбрасываешь
  щелчком/флик-драгом, следующая всплывает, цикл бесконечный.
- `mood10.html` — **Karuselis**: 3D-барабан-коверфлоу из двух колец;
  драг крутит, колесо листает, передняя грань крупнее.

### Экспериментальные — WebGL / canvas (mood11–mood15)

- `mood11.html` — **Šķidrums**: вся сетка работ (атлас-текстура) течёт как
  жидкость — стоячие волны + курсор-линза + хроматические края. WebGL, один квад.
- `mood12.html` — **Punkti**: работы как типографский растр из точек на крафте,
  медленный кроссфейд; курсор-лупа показывает чистое цветное фото. Canvas 2D.
- `mood13.html` — **Spogulis**: вращающийся калейдоскоп складывает атлас в
  зеркальную розетку; курсор X — число сегментов (6–16), Y — зум. WebGL.
- `mood14.html` — **Atbalss**: видео-обратная связь — работа бесконечно
  затягивается в спиральный тоннель эха (canvas рисует сам себя со сдвигом);
  курсор рулит точкой схода, клик переключает зум наружу⇄внутрь. Компоновка
  `source-over` (не аддитивная), поэтому яркость не может превысить самый
  яркий пиксель фото — тоннель никогда не выцветает в белый экран.
- `mood15.html` — **Drumstalas**: каждая работа собирается из ~8000 цветных
  частиц (плотность ∝ яркости), держится и рассыпается в следующую; курсор
  расталкивает рой. Открывается на самой контрастной работе. Canvas 2D.

Общая заметка по превью: встроенный Launch-превью отдаёт вьюпорт 0×0 до
материализации таба и тикает rAF только во время снятия скриншота, поэтому
canvas/WebGL-доски и эффект эха на первом кадре выглядят пустыми — на реальном
Chrome рисуются сразу. Первая сборка везде обёрнута в `onViewport()`.
